# enumerable_testing

This repo has rspec for custom nine enumerable functions in ruby.

## Installation

#### Requirement
* Ruby 2.5.0+
* Gem 3.0.4+

#### Run

##### Clone
```  
git clone https://github.com/MouhaDiouf/enumerable_testing 
cd enumerable_testing
```

##### Bundle install

```
bundle install
```

##### Run test

```
rspec
```
